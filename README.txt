해당 프로젝트는 실행 후 MainScene에 접근하게 됩니다.

MainScene에는 이동 및 상호작용이 가능합니다.
해당 씬에서는 스페이스바로 점프기능도 구현해봤습니다.

왼쪽 위 포탈에서 F키로 상호작용을 하면 원래 위치로 돌아갈 수 있습니다.

다른 Scene으로 이동할 때는 위에 있는 집으로 들어가면 Trigger로 특정 Scene으로 이동하게 만들었습니다.

MainScene에서 접근 가능한 미니 게임은 현재 플래피버드 하나입니다.
-SceneName: PlaneGameScene
점프: 마우스 클릭 OR 스페이스바 OR 'F'

 플레이어가 앞으로 가서 장애물과 부딪히는 기존 강의와는 다르게 배경과 장애물이 플레이어 왼쪽을 향해 이동시키며 Trigger에 닿으면 재배치하는 방식으로 만들어 봤습니다.
 이 방식은 카메라를 고정시킬 수 있으며 플레이어의 이동은 Y축으로 점프만 구현하면 된다는 이점이 있었습니다.
 하지만 그 외의 모든 오브젝트를 왼쪽으로 이동시켜야 한다는 단점이 있어 어떤 방식이 더 좋은지는 잘 모르겠습니다.
 그래도 플레이어의 X좌표가 고정될 수 있다는 이점을 이용하여 장애물 오브젝트를 오른쪽에 쌓아둔 후 플레이어에게 특정 주기마다 던지는 식으로 난이도 조절을 간편하게 할 수 있도록 바꿔봤습니다.
5점 -> 10점 -> 20점 -> 50점이 될 때마다 장애물을 던지는 주기를 줄이고 배경의 속도를 높여 난이도를 높여봤습니다.
